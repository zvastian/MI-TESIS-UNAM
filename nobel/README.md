# Nobel Atlas

Carpeta consolidada de la visualización Nobel.

Incluye:
- `nobel_map.html`: visualización Sigma del Atlas Nobel.
- `static/`: payloads JSON usados por la demo estática.
- `outputs/raw/`: datos crudos descargados desde Nobel Prize API.
- `outputs/processed/`: datos procesados para construir el atlas.
- `outputs/analysis/`: resúmenes, auditorías y diagnósticos.
- `outputs/cache/`: cache de traducciones de motivaciones.
- `outputs/embeddings/`: embeddings generados para motivaciones Nobel.
- `scripts/`: scripts usados para construir, traducir y auditar el atlas.

Nota: algunos archivos son generados y podrán filtrarse o moverse a storage externo más adelante.
